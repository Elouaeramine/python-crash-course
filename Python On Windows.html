Python On Windows

Installing Python on Windows

  • Go to https://www.python.org/downloads/
  • Click on Download Python 3.8.5 ( Or The latest version )
  • After the Download, Execute the file
  • Make sure the Install launcher for all users (recommended) and the Add Python 3.8 to PATH checkboxes at the bottom are checked.
  • Go through with the installation
  • Close the Python Setup

Python should be installed now

  • To make sure the installation is complete. Open a command prompt and type python
  • You should see something like this

Running 'Hello World !' in a Terminal

  • Python can be executed directly from the command line / Python IDLE
>>> print('Hello World !')

Executing Python

  • Another way of working with Python is through files
  1. We create a file with the extension .py
  1. We save the file
  1. We then run the file with the command
python myfile.py

Installing a Text Editor

  • To get through this crash course. You will need a Text Editor
  • You can choose a Text Editor of your preference
  • Most people use one of these Text Editors ( Atom, Sublime Text , Visual Studio Code )
  • We will be using Visual Studio Code in This Course
